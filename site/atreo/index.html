ATREO

E-Commerce | RWD | Front-End

PROJECT DESCRIPTION

The Atreo website was the e-commerce and online marketing launch of the apparel brand Atreo. Built off the Shopify platform, this site gives the user an easy and intuitive design to manage their e-commerce business. This site features a full viewheight landing page with a modern minimalistic feel. My responsibilties for this project were to bring the design from concept to reality, and to set up and pass the management of the platform to the customer.

THE WEBSITE

The shopify platform uses mobile first deisgn for a website that functions and looks good on every device.
